# sw-single-page-cv-latex-templates

單頁 CV / 履歷的 LaTeX 模板。  
LaTeX templates for single-page CV and resume documents.

## 概述 | Overview

專業的 LaTeX 模板，設計用於製作簡潔的單頁 CV 與履歷文件，針對清晰度與易於自訂進行最佳化。  
Professional LaTeX templates designed for creating clean, single-page CV and resume documents. Optimized for clarity and ease of customization.

## 如何執行 | How to Run

1. 確認已安裝 LaTeX 發行版（如 TeX Live、MiKTeX）。  
   Ensure you have a LaTeX distribution installed (e.g., TeX Live, MiKTeX).

2. 編譯文件 | Compile the document:
   ```bash
   pdflatex main.tex
   ```

3. 或使用偏好的 LaTeX 編輯器（如 Overleaf、TeXmaker、VS Code with LaTeX Workshop）。  
   Or use your preferred LaTeX editor (e.g., Overleaf, TeXmaker, VS Code with LaTeX Workshop).
